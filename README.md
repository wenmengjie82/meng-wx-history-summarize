# meng-wx-history-summarize

This repository contains the `meng-wx-history-summarize-skill` skill for authorized local WeChat PC 4.x history recovery, export, scanning, and report generation.

No WeChat databases, keys, raw chat exports, personal paths, chatroom IDs, or generated private reports are included.
