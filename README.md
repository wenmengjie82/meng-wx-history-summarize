# 微信聊天记录总结 Skill

这个仓库包含 `meng-wx-history-summarize-skill`，用于在本地处理用户本人有权访问的微信 PC 4.x 聊天记录：数据库恢复、群聊扫描、指定群导出、日报和月报生成。

请仅用于处理自己有权访问的数据。运行过程产生的本地文件由使用者自行管理。
